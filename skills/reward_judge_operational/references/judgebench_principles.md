# JudgeBench Principles

For answer-key tasks, extract final answers from both responses and use the visible reference as the strongest signal.
For subjective tasks, judge instruction satisfaction and error avoidance before prose quality.

Use a two-pass check:
1. Final-answer pass: what exact option, number, code result, or requested output did each candidate submit?
2. Support pass: does the candidate's reasoning, format, and visible reference evidence support that final answer?

If the visible reference is noisy or ambiguous, do not let it override every other signal. Prefer the response that best satisfies the prompt under the visible evidence.
