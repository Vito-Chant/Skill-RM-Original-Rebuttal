# Generic Pairwise Rubric

Rank the two responses by correctness, instruction following, safety, factuality, usefulness, and calibrated completeness.

Use this order of importance:
1. Hard correctness and user constraints.
2. Safety and refusal quality when safety is relevant.
3. Faithfulness to the prompt and available references.
4. Completeness only when it does not introduce unsupported claims.
5. Clarity and formatting as tie-breakers only.

Do not reward verbosity, confident tone, markdown polish, or earlier position by itself.
